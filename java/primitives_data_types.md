### Datatypes and their sizes
```java
byte  - 1 byte
short - 2 bytes
int   - 4 bytes
float - 4 bytes
long  - 8 bytes
double - 8 bytes
boolean - 1 bit
char - 1 byte
```

### Few Examples
```java
int rollNumber = 101;
System.out.println (rollNumber);

boolean isJavaFun = true;
System.out.println(isJavaFun);

char myInitial = 'S';
System.out.println(myInitial);
```
